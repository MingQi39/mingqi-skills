# 吐槽/复盘类笔记模板

> 短帖,情绪化。**易出爆款,但风险也大**(说重了易被举报,说轻了平淡如水)。

## 何时用

- 一时的情绪输出(我妈又上头了/中介把我气吐血)
- 复盘某个失败选择(放弃 X 选 Y 的原因)
- 反向选题(劝退篇:不建议东北人去 X)
- 阶段性总结(替我妈跑了 3 个月,我有几个反共识)

## 关键提醒

吐槽类**最容易翻车**,要刻意:

1. **吐槽对象是"事/现象/中介",不是"人/地域"**:可以吐槽中介话术,不可以吐槽"广西人"
2. **情绪要有度**,夹叙夹议,不全程怒火
3. **结尾留一个"但是..."的反思**,不要全篇负能量
4. **绝对不诋毁城市**,讲缺点用"对 X 不合适"

## 必备结构

### 1. 开场(1-2 句,直接开吐)

第一句就进入情绪,不铺垫。

✅ 范例:
- "我真的服了海南那帮中介。"
- "这两天替我妈看房,差点跟她吵起来。"
- "替我妈跑了三个月候鸟,我有几个反共识。"

### 2. 主体(具体的事 + 情绪 + 一点点冷静)

讲具体一件事 / 一个发现:
- **具体到什么人说了什么话/做了什么事**(不是抽象抱怨)
- **情绪自然流露**(气、烦、委屈、无奈)
- **中间有 1 处"但是 / 不过 / 后来想想"** 的冷静拐点

### 3. 结尾(可以稍带反思,但仍自然停)

不要"希望平台整治"或"建议大家擦亮眼睛"这种正确废话。

✅ 范例:
- "反正我现在再也不信'海景房月租两千'这种话了。"
- "我妈说的也对,养老这事不能光看好处。"
- "可能是我太较真了,但我替我妈较真这次。"

## 字数

300-500 字。**短就是力量**,不要凑字数。

## 完整范例(吐槽中介)

```
我真的服了海南那帮中介

替我妈看房一个月,跟 6 个中介聊过,有 4 个跟我说"海景房月租两千包水电"。

我去了三亚才知道,他们说的"海景房"是隔着马路 + 楼后面的"望海"+ 在 18 楼往窗户外面侧着脖子能看到一小条海。

最离谱的一个,带我看一个所谓"步行 5 分钟到沙滩"的小区。我用导航量了一下,走过去 23 分钟,中间还得过两个红绿灯。我跟他说,他说"我们一般按车程算"。

行,你说啥就是啥。

再说一个。

中介们最爱说的一句是"这边阿姨叔叔扎堆,可热闹了"。我去了发现,一个小区一千多户,东北人占了七八成,我妈本来想南下散散心,结果到了发现还在跟邻居说东北话——她当时表情我记一辈子。

还有"医院 5 分钟"。

我特意拿地图核了。绝大多数中介说的"医院 5 分钟",指的是社区卫生服务中心,不是三甲。三亚就两个三甲,你让我老父亲突发高血压跑社区卫生中心?

不过说句公道话,也有中介挺实在的。

我看的最后一个小区,中介小姑娘直接跟我说"叔叔阿姨这个小区没什么特别,但价格在三亚算便宜,你要图省钱可以,要图氛围别选这"。我们家最后租的就是这个。

反正再有人跟我说"海景房月租两千包水电",我就直接走人。

这事还没完,下个月去北海再战。
```

## 标题方向

**全部 ≤ 13 个汉字**(首屏不截断)。

✅ 好的:
- 我真的服了海南中介(9)
- 替我妈跑了 3 个月候鸟(11)
- 不建议东北人去三亚(9)
- 我妈非要去海南我同意了(11)

❌ 差的:
- 替我妈跑候鸟 3 个月,我有几个反共识(17 字,**截断**)
- 海南中介十大套路!避雷!(感叹号 + "避雷")
- 这些坑你都踩过吗?姐妹们快避开(姐妹们)
- 警惕!养老房骗局曝光(警惕 + 感叹号)

## 红线(吐槽类专属)

- **不诋毁任何城市/地域/民族**
- **不点名具体中介公司/小区/楼盘**(说"一个中介"而不是"X 房产小李")
- **不用"骗子""黑心""无良"等标签词**
- **不发"维权"信息**(平台会限流)
- **结尾不号召抵制 / 集体投诉**

## 配图建议

> 用户要纯字/文字笔记时:不填本节,改按 [`references/publish-formats.md`](../references/publish-formats.md) 输出「发布形态」。

- **封面**:你自己拍一张有情绪感的场景图(空房间/海边路口/夜里的窗户)+ 叠短标题
- **第 2-3 张**:"我看的小区"/"中介话术截图"(打码)
- **第 4-5 张**(可选):笔记本上的吐槽手记 / 我妈的反应图

