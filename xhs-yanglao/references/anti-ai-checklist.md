# 人味自检清单(发布前 MUST 全过)

任何笔记生产完成后,逐条检查。**只要有一条不过,重写。**

## 一、词汇红线(任一命中 = 重写)

正文里**绝对不能出现**:

- [ ] 没有"姐妹们""家人们""宝子""绝绝子""巨""神仙""yyds"
- [ ] 没有"干货""必看""强烈建议""码住""建议收藏""一定要"
- [ ] 没有"今天给大家分享""希望对大家有帮助""感谢观看"
- [ ] 没有"首先...其次...最后...""综上所述"
- [ ] 没有"Step 1 / Step 2"等编号(除非真在写流程教学)
- [ ] 没有 ✨ 🔥 ❤️ 等装饰性 emoji 堆叠(单个偶尔可以)
- [ ] **标题 ≤ 13 个汉字**(首屏不截断;数字 / 英文按 1 字算)

## 二、结构红线

- [ ] 开头第一句是**具体场景或情绪**,不是观点/总结
- [ ] 结尾**没有**总结、升华、号召、问"你怎么看"
- [ ] 中间至少有 **1 处"跑题"** 的生活细节(我妈在旁边/我妈又念叨什么/桌上的茶等)
- [ ] 数据**没有**用列表 + bullet 罗列,而是夹在叙述里
- [ ] 段落分隔自然,不是模板感的工整三段
- [ ] **没有为节奏强行断行**(一段就是一段连续的话,5-8 段成文,段间只空一行)
- [ ] **标点全部全角**(,。;:?——……),引述别人说的话**不加引号**;数字 / 英文两侧加半角空格(详见 [`SKILL.md`](../SKILL.md) 「排版与标点规范」)

## 二·五、Sequel 承接自检(承接评论热门题时 MUST,见 `sequel-strategy.md`)

- [ ] **类型是故事/避坑**,不是纯教程/横评(医保、三城也须叙事包装)
- [ ] 开篇**不是**「评论区问太多了」「接上条」
- [ ] 与 #19 正文比对:**无连续 8 字以上相同段落**(三亚三千五/万宁 23 分钟等金句禁复用)
- [ ] 正文中段有**具体提问钩子**,不是文末「你怎么看」
- [ ] Phase 2 默认 **文字笔记**,非脚本全套配图

## 三、人味必备(必须包含 ≥ 3 项)

逐条勾选,**至少命中 3 条**:

- [ ] 有"我也搞不懂/我也烦/我也犹豫/我也吵架"这类自我暴露
- [ ] 有具体时间锚点("上周三""那天晚上""上个月")
- [ ] 有口语虚词("吧""呗""说实话""挺""那种""怎么讲")
- [ ] 有情绪波动(烦躁/委屈/释然/吐槽)
- [ ] 有具体细节(具体数字/对话/物件,而不是抽象描述)
- [ ] 有"我妈"出现 ≥ 3 次(本账号叙事**只写我妈**;不写「我爸/爸妈/父母」指代自家,见 [`persona.md`](persona.md))

## 四、信息红线

- [ ] 政策/数据如果不确定,行文里有"我查的时候是这样,你最好自己再核对一下"
- [ ] 没有替任何房产/养老社区/保险/中介背书
- [ ] 没有"包治""绝对""稳赚""治好""一定"等绝对化用语
- [ ] 讲城市缺点时用"对 X 人群可能不合适",没用"这地方不行"

## 五、幽默基调自检(详细规则见 [`humor.md`](humor.md))

- [ ] 笑点**没有**落在父母身上(不拿父母身体/钱包/认知开涮)
- [ ] 笑点**没有**落在地域/民族/方言上
- [ ] 一段里没有连用 3 个以上梗(不是脱口秀稿)
- [ ] 没用网络烂梗(u1s1 / 绝绝子 / 谐音梗 / 表情包用语)
- [ ] 结尾**没有**抖包袱,是平收 / 留白 / 未完成的画面
- [ ] 笑过之后有真情绪垫底(累 / 心疼 / 烦 / 犹豫),不是空段子

## 六、最终三问(全部回答"是"立刻重写)

1. **如果我把这段发我闺蜜微信,她会觉得奇怪吗?**
2. **里面有没有任何一句话,我现实生活中绝对不会说?**
3. **这段读起来,像不像在"运营"一个号?**

任一"是" → 重写。

## 七、发布形态自检(二选一)

先确认本条用哪种形态(见 [`publish-formats.md`](publish-formats.md)):

### A. 配图模式

- [ ] 封面**不是**数据表/对比表(数据图放第 2-3 张)
- [ ] 封面有生活感(场景物件/暖色调,不是花花绿绿的字图)
- [ ] 封面叠字 ≤ 12 字,字号大,黑底白字或白底黑字,不用花体

### B. 文字笔记模式(**本账号 Phase 2 默认**;用户明确不要配图时)

- [ ] 使用 App「写文字」/「文字笔记」,非硬塞图文笔记只填字
- [ ] §2 正文为**纯文本**:无 `**` / Markdown;逗号为全角 `，`(非 ASCII `,`)
- [ ] §2b 加粗清单 **3–6 条**;至少 1 条是中段评论钩子
- [ ] 已提醒用户:先粘贴正文,再按清单 App 内手动加粗
- [ ] 系统封面模板选**简洁**款(避免花字/渐变营销风)
- [ ] 标题仍 ≤ 13 字;标签 5–8 个
- [ ] **本节 A 三条不适用**,勿因没自拍而阻塞发布
