# 信源备查清单

**写笔记前**,凡涉及政策/数据/具体地点,先到下面的信源核对。**不查不写**。

---

## 政策类(社保/医保/退休)

| 信息类型 | 必查来源 | 怎么查 |
|---|---|---|
| 退休年龄/养老金计算 | 国家社会保险公共服务平台 (si.12333.gov.cn) | "养老金测算"在线工具 |
| 异地医保备案 | 国家医保服务平台 APP | 首页"异地就医备案" |
| 各市具体退休细则 | 当地人社局官网 | 搜"XX市人社局" |
| 各市医保政策 | 当地医保局官网 | 搜"XX市医保局" |
| 不确定时打电话 | 12333(全国) / 当地区号+12333 | 5 分钟搞清楚比写错强 |

---

## 城市数据类

| 信息类型 | 必查来源 | 备注 |
|---|---|---|
| 房租 | 贝壳找房 / 链家 / 安居客 | 搜具体小区,看实际挂牌 |
| 长租/月租公寓 | 携程长租 / 美团民宿(长租筛选) | 候鸟群体常用 |
| 买房房价 | 贝壳房产 / 中国房价行情(home.fang.com) | 看二手成交,不看新盘广告 |
| 月均生活成本 | 数据帝(shujudi.com) / 用户实测帖 | 交叉验证,不只信一个源 |
| 气候(月均温/湿度) | 中国天气网历史数据 | 看至少 3 年平均 |
| 三甲医院 | 各市卫健委官网 | 搜"XX市三级医院名录" |
| 公交/地铁 | 高德/百度地图 | 直接搜路线 |

---

## 体验/口碑类

| 信息类型 | 必查来源 | 备注 |
|---|---|---|
| 真实候鸟经历 | 小红书搜"在 XX 养老""候鸟 XX" | 看 1 年以上经历的帖 |
| 当地生活体验 | 知乎"在 XX 养老是什么体验" | 长答案更可信 |
| 中老年视角 | 抖音/快手搜"我在 XX 养老" | 真人 vlog 信号最强 |
| 投诉/吐槽 | 黑猫投诉 / 小红书"XX 避坑" | 反向校验 |

---

## 房产类(谨慎,涉及变现合规)

| 信息类型 | 必查来源 | 备注 |
|---|---|---|
| 各地买房限购政策 | 地方政府官网公告 | 政策每年变,**不引用过期信息** |
| 二手房成交价 | 贝壳"成交"数据 | 比挂牌价真实 |
| 候鸟公寓口碑 | 小红书 + 知乎 + 闲鱼出租 | 看真实租客反馈 |

⚠️ **不替任何具体楼盘/小区背书,不分享楼盘联系方式,不接中介软广**(除非合规接广,且必须标记)。

---

## 不确定时怎么办

1. 写不出 → **不写**,换选题
2. 数据冲突 → 在文末「核对清单」明确写"我查到 X,但有些资料是 Y,你最好自己再确认"
3. 政策可能变了 → 加一句"我查的时候是这样,你最好自己再核对一下"
