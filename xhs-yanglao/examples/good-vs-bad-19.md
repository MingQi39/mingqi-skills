# 好/坏笔记对照(以选题 #19 为例)

## 选题
《我替我妈考察了 8 个城市,最后选了这个》

---

## ❌ 坏版本(典型 AI 味)

```
北方人去海南养老,这 3 个真相中介不会告诉你

大家好!最近很多朋友问我海南养老怎么样,今天就来给大家深度分析!✨

🔥 真相一:房租比想象贵
三亚淡季月租约 3000-4000 元,旺季可达 6000-8000 元。
海景房月租 5000+ 起步,远超北方人预期。

🔥 真相二:夏天比东北热
三亚 7-8 月平均气温 28°C,湿度高达 80%,
体感温度可达 35°C 以上,北方人难以适应。

🔥 真相三:医疗资源有限
三亚仅有 2 家三甲医院,
对慢病老人不太友好。

姐妹们!选择养老地一定要擦亮眼睛~
你妈想去海南吗?评论区聊聊~
觉得有用记得点赞收藏哦!❤️

#异地养老 #候鸟养老 #海南养老 #三亚 #必看 #干货
```

### 为什么差
- "大家好""今天给大家分享":AI 味开头
- "🔥 真相一/二/三":营销号工整结构
- 数据用 bullet 罗列,没有人味缓冲
- "姐妹们""觉得有用记得点赞收藏":号召式结尾
- 没有任何"我家"细节,完全是攻略口吻
- 标签里有"必看""干货" → 平台 AI 内容识别会扣分

---

## ✅ 好版本(人味)

```
我妈非要去海南,我劝了三个月

我妈今年 62,东北人,过完这个冬天嚷着要去三亚。我说你先别急,我去看看。

上个月去了一趟。

第一天我就发现不对。出租车司机跟我说,你们东北人来这边租房,十一月前订才有好价。我妈想象中的那种"海景房月租两千",淡季也得三千五往上,旺季翻倍我都不敢想。

第二天去医院附近转了下。三亚就两家三甲,离我看的几个小区都不近。我妈这两年血压不太稳,这事我也没敢跟她细说,怕她又上头。

然后是天气。我以为冬天去就行了——我妈也是这么想的。结果碰上一个回南天,被子摸上去都是潮的。当地人说这种天气一年好几次,我看我妈那张脸,有点开始动摇了。

但说实话,海边晚上散步是真的舒服。我妈那天在沙滩上走了一个小时没说话,回来跟我说"老李头来这住过两冬,人都精神了"。

我没接话。

这事还没完,可能下个月再去一次北海看看。

反正不是中介说的那么简单。

#异地养老 #候鸟养老 #退休生活 #我妈的养老地图 #海南养老
```

### 为什么好

| 维度 | 体现 |
|---|---|
| 开场是场景 | "我妈今年 62,东北人,过完这个冬天嚷着要去三亚" |
| 自我暴露 | "这事我也没敢跟她细说,怕她又上头" |
| 时间锚点 | "上个月去了一趟""第一天""第二天""那天" |
| 跑题/副线 | "老李头来这住过两冬,人都精神了" |
| 情绪起伏 | 期待 → 失望 → 动摇 → 但是海边真舒服 → 我没接话 |
| 口语虚词 | "说实话""怎么讲""挺""那种""有点" |
| 数据夹叙述 | "海景房月租两千" 是引用我妈的想象,具体数字夹在叙事里 |
| 结尾自然停 | "这事还没完,可能下个月再去一次北海" |
| 我妈/我家 | "我妈"出现 8 次以上(本账号叙事**只写我妈**) |
| 标签克制 | 5 个,无"必看/干货" |

---

## 关键学习点

1. **同一选题,角度差很多**。坏版本是"信息分发",好版本是"我有事想说"
2. **数据没少**(三亚两家三甲医院、回南天、租金虚高都讲了),但是**用叙事包起来**
3. **结构上故意不工整**——主体段长短不一,有的 4 行有的 1 行(我没接话)
4. **没有"求关注/求收藏"**,但因为内容真实,反而更易被收藏

## ❌ Sequel 失败对照:#1 与 #19 自我重复(2026-06-08)

#1《三亚海口万宁怎么选》各 **~100 阅**。正文大段与 #19 相同:

```
#19: 三亚最贵。我看的那几个小区,淡季月租也得三千五往上…
#1:  三亚最贵…淡季月租三千五往上…(几乎 verbatim)

#19: 万宁是中介推得最猛的一个…步行 5 分钟…实际走 23 分钟…
#1:  万宁是中介推得最猛的…(几乎 verbatim)
```

### 为什么惨

- 老读者:#19 已看过,划走
- 新读者:缺 #19 的 8 城旅程新鲜感,像摘要帖
- 算法:同账号相似度风险(`algorithm.md` §七.5)
- 形态:脚本配图 vs #19 纯字 CTR 19.6%

### Sequel 正确写法

| 话题 | ❌ #1 做法 | ✅ 应做 |
|---|---|---|
| 三城 | 横评复述租金 | 《陪我妈看医院那天》— 只写医院场景,不提三千五/23 分钟 |
| 医保 | 教程《怎么备》~100 阅 | 《医保局排了一上午》— 步骤夹在排队故事里 |
