# 选题库模板

> 用户本地维护，**默认不入 git**（可放在 `state/topic-pool.md`）。AI 推荐选题时结合 `hot-topics.md` 检索，不要只机械翻本表。

---

## 怎么用

1. 按账号赛道填 20–30 个候选题
2. 每条标注：类型（故事 / 对比 / 实操 / 吐槽）、是否已发、备注
3. 评论区收集的高频问句 append 到文末
