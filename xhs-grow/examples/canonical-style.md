# 排版基准范本

> 人味笔记的标点 / 换行 / 引述风格基准。用户可将自己的改定稿替换本节正文，作为该账号的唯一排版参照。

---

## 正文（基准范本）

```
上周三下午，我去办一件事，以为半小时能搞定，结果在门口排了俩小时。

窗口的人说，App 上就能办。我当场打开又关了一次，怕点错。旁边有个阿姨一直在念叨，说她们那儿不用排队，我装没听见。

办完出来天已经黑了。我没跟谁说这事，就觉得自己挺蠢的，明明可以先打电话问一句。

反正下次还得再跑一趟，先把材料凑齐吧。
```

---

## 为什么是基准

### 标点

- 全部全角：`，` `。` `；` `：` `？`
- 引述别人的话**不加引号**，直接写在句里

### 数字 / 英文

- 中文与数字 / 英文之间加半角空格：`排了 2 小时`（按用户习惯可统一）

### 段落

- 段内不为节奏强行断行
- 每段 1–3 句，全文 5–8 段
- 段间只空一行，无标号、无 bullet

### 开头 / 结尾

- 开头：具体场景（上周三下午…）
- 结尾：自然停（反正下次…），不号召、不升华
